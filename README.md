# SmartDocs - AI Document Chat

A web app to upload PDFs (like resumes) and chat with them using AI.  

## Features
- Upload PDF files
- Extract text from documents
- Ask questions about the content
- Chat interface with history
- Works with resumes and similar documents

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express, Multer
- Database: SQLite (optional for storage)
- OCR: Tesseract.js
- AI: OpenAI API
